# AI Habitat OS - Ecosystem Map

This file documents the routing network, agents, and nodes of the Pickaxe Agent City / AI Habitat OS.

## 1. Agent Roles

- **Research Agent:** Data science, market signals, pattern analysis.
- **Trading Agent:** Liquidity, order flow, execution research only.
- **Builder Agent:** Infrastructure, code, deployment.
- **Risk Agent:** Risk control, stress testing, exposure analysis.
- **Memory Agent:** Archive, bookmarks, knowledge graph.
- **QA Agent:** Validation, system testing, quality gate.
- **Strategy Agent:** Simulation, decision engine, battle plans.
- **Recon Agent:** Intel gathering, threat detection, opportunity scan.
