# UI Principles

The interface should feel like a luxury cyberpunk command center: dark, precise, reactive, and calm.

## Direction

- Premium futuristic, not cartoonish
- Dense enough for traders, not cluttered
- Cinematic contrast, readable text, restrained neon
- Living system behavior through subtle motion
- Every screen should show operational value immediately

## Components

- Use cards for individual instruments, agents, alerts, and logs.
- Use full-width bands for major sections.
- Use icons or compact labels only where they improve scanning.
- Keep border radius tight.
- Prefer clear panels and readable metrics over decoration.

## Motion

Motion should imply intelligence:

- agent pulses
- status shimmer
- traveling data packets
- active connection paths
- progress bars
- live logs

Motion should never make trading data harder to read.
