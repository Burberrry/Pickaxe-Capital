# Vision Map Architecture

The Vision Map is the project brain.

## Domains

- Brain Map: inputs, tools, markets, devices, and research categories
- Monitor Room: live market and catalyst categories
- Learning System: psychology, mental models, trader stories, formulas, journals
- Business Map: TTT, physical ventures, goods, websites, apps, import/export, revenue streams
- Catalyst Memory: political, macro, public-market events
- Agent Habitat: specialist agents and collaboration paths

## Rule

The Vision Map should become more structured over time without becoming bloated. Add only durable concepts that improve future decisions.
